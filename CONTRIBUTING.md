# Contributing to PointRealm

Thanks for your interest in contributing! This document outlines the process
for contributing changes.

## Getting Started
1. Fork the repository and create your branch from `main`.
2. Ensure you have .NET 10 SDK and Node.js installed.
3. Install frontend dependencies in `src/PointRealm.Client`.

## Development Guidelines
- Keep changes focused and well-documented.
- Run `dotnet build` and relevant tests before submitting.
- Follow the repository code style and conventions.

## Pull Request Process
1. Update documentation as needed.
2. Ensure all checks pass.
3. Request review from maintainers.

## Code of Conduct
Please read and follow the [Code of Conduct](CODE_OF_CONDUCT.md).
